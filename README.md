# Python port tester

## Usage

### host.py usage

`python3 host.py -m UDP -p 666`

`-m` or `--mode` takes udp or tcp
`-p` or `--port` takes a port number

NOTE: This script requires admin privileges!

### client.py usage

`python3 client.py -m UDP -p 666`

`-m` or `--mode` takes udp or tcp
`-p` or `--port` takes a port number

NOTE: This script requires admin privileges!

## Additional comments

Please always run the host before the client.
